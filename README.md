# FlowDash Mobile

A native iOS dashboard for remote management of n8n automation workflows.

## Features

- **Remote Workflow Control**: Securely monitor and manage n8n workflows from your iPhone
- **Execution History**: View run logs and status in real-time
- **Toggle Workflows**: Enable/disable workflows on the go
- **Dark Theme**: Streamlined interface optimized for mobile use
- **Secure Authentication**: Safe API key storage and connection

## Tech Stack

- Swift
- SwiftUI

## Design

## Requirements

- iOS 16+
- Xcode 15+
- n8n instance with API access
